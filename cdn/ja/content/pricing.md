## 無料プラン(オープンソース)
- **対象:** 個人プロジェクト、オープンソースソフトウェア、ホビイスト。
- **料金:** 0 ドル/月 — 永久無料、クレジットカード不要。
- **帯域幅:** 10 GB/月。超過するとアセットの CDN 配信は停止します(ファイルは git に残ります)。
- **アセット:** 無制限の静的ファイル(PNG、WebP、AVIF、SVG、ICO)。
- **自動最適化:** プッシュされたすべての PNG/JPEG が自動的に WebP(品質 80)と AVIF(品質 65)に変換されます。
- **配信:** Cloudflare の 300 以上のエッジ PoP からの TTFB 100ms 未満。不変キャッシュ(1 年間の max-age)。
- **制限:** カスタムドメインなし、画像変換 API なし、分析ダッシュボードなし。

## Pro プラン
- **対象:** 商用プロジェクト、スタートアップ、トラフィックの多いウェブサイト。
- **料金:** 29 ドル/月(または 278 ドル/年 — 20% 割引)。
- **帯域幅:** 100 GB/月込み。超過分: 0.05 ドル/GB を周期末に請求。
- **画像変換 API:** URL パラメータでオンザフライにリサイズ、トリミング、変換します:
  ```
  https://cloudcdn.pro/project/image.png?w=800&h=600&fit=cover&format=auto&q=80
  ```
- **形式ネゴシエーション:** ブラウザの `Accept` ヘッダーに基づいて AVIF/WebP を自動配信。
- **カスタムドメイン:** 自動 SSL プロビジョニング付きで最大 5 ドメイン。
- **優先サポート:** 24 時間以内の返答保証付きメールサポート。
- **SLA:** 99.9% のアップタイム。違反時はしきい値を 0.1% 下回るごとに 10% のサービスクレジット。
- **分析:** 帯域幅、リクエスト数、キャッシュヒット率、形式の分布。
- **14 日間の無料試用**でフルアクセス。クレジットカード不要。

## Enterprise プラン
- **対象:** グローバルプラットフォーム、メディア企業、大規模 EC サイト。
- **料金:** カスタム価格 — sales@cloudcdn.pro までお問い合わせください。
- **帯域幅:** 専用エッジ割り当てで無制限。
- **フル API スイート:** Pro のすべての変換に加えて、バッチ処理、Webhook 通知、プログラマティックアップロード。
- **カスタムドメイン:** ワイルドカード SSL 付きで無制限。
- **専任サポート:** 担当アカウントマネージャー、プライベート Slack チャンネル、1 時間応答 SLA。
- **SLA:** 99.99% のアップタイム。金銭的クレジット: 99.99% 未満で 25%、99.9% 未満で 50%、99.0% 未満で 100%。
- **セキュリティ:** SSO/SAML、12 か月保持の監査ログ、IP 許可リスト。
- **分析:** アセット別、地域別、形式別の内訳を含むリアルタイムダッシュボード。
- **コンプライアンス:** GDPR DPA 提供可能、データレジデンシーオプション(EU/米国)。

## 課金
- 既定では月次請求。年次請求では 20% お得です。
- 無料プラン: クレジットカード不要、試用期限なし。
- Pro 試用: 14 日間、全機能アクセス、クレジットカード不要。
- 超過分: 周期末に請求、サイクル途中でのサービス中断は決してありません。
- いつでもキャンセル可能。長期契約なし。キャンセル後もデータには 30 日間アクセスできます。

## 公正利用ポリシー
CloudCDN は静的アセット(画像、アイコン、フォント、ドキュメント)の配信向けに設計されています。動画ストリーミング(25 MB 超のファイル)、アプリケーションバイナリの配布、ファイルホスティングを目的としていません。公正利用を超えるアカウントには適切なプランについてご相談するためにご連絡します。事前通知なしに停止することは決してありません。
