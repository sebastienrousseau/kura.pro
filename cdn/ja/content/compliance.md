# CloudCDN — コンプライアンスとプライバシー

## データ処理

### CloudCDN はどのようなデータを扱いますか?
CloudCDN は静的ファイル(画像、アイコン、フォント)を配信します。個人ユーザーデータの処理、保存、送信は行いません。データの流れは次のとおりです:
1. ブラウザが静的ファイルの URL をリクエストします。
2. Cloudflare のエッジがキャッシュされたファイルを配信します。
3. 標準的な HTTP ログ(IP、タイムスタンプ、URL、user-agent)が生成されます。

### データはどこに保存されますか?
- **ソースファイル:** GitHub リポジトリ(GitHub/Microsoft により米国でホスティング)。
- **エッジキャッシュ:** Cloudflare の 300 以上のグローバル PoP。パフォーマンスのためにキャッシュコピーが世界中に分散されます。
- **AI コンシェルジュのデータ:** 会話は保存されません。チャットウィジェットはメモリ上のセッション状態のみを使用し、ユーザークエリのサーバー側ロギングは行いません。
- **レート制限カウンター:** Cloudflare Workers KV に保存(集計値のみ、個人データなし)。

## GDPR コンプライアンス

### ステータス
CloudCDN は GDPR に準拠しています。インフラプロバイダーである Cloudflare は、以下を通じて GDPR 準拠を維持しています:
- EU-US Data Privacy Framework の認証。
- 国際データ移転のための標準契約条項(SCC)。
- リクエストに応じて利用可能なデータ処理契約。

### データの最小化
- CloudCDN はクッキーを設定しません。
- ユーザートラッキングや解析ピクセルはありません。
- 個人データの収集、保存、処理は一切行いません。
- HTTP アクセスログは Cloudflare がそのプライバシーポリシーに従って処理します。

### データ主体の権利
CloudCDN は個人データを収集しないため、アクセス、訂正、削除すべき個人データは存在しません。アセット(写真など)に個人データが誤って含まれていると思われる場合は、support@cloudcdn.pro までご連絡ください。

### DPA(データ処理契約)
Enterprise のお客様は正式な DPA をリクエストできます。sales@cloudcdn.pro までご連絡ください。

## CCPA / CPRA(カリフォルニア州)
CloudCDN はターゲット広告のために個人情報を販売、共有、使用することはありません。個人データを収集しないため、オプトアウトの仕組みは不要です。

## SOC 2 / ISO 27001
CloudCDN は Cloudflare のインフラを活用しており、Cloudflare は以下を維持しています:
- SOC 2 Type II 認証。
- ISO 27001 認証。
- PCI DSS Level 1 準拠。
これらの認証は、CloudCDN が使用するエッジ配信インフラを対象としています。

## セキュリティ対策
- **転送時の暗号化:** すべての接続で TLS 1.3。
- **DDoS 保護:** すべてのプランで Cloudflare の自動 DDoS 軽減。
- **WAF:** すべてのエンドポイントで Cloudflare Web Application Firewall が有効。
- **ボット軽減:** Cloudflare Bot Management がスクレイピングと不正使用から保護。
- **署名付きコミット:** すべてのアセット変更には暗号学的検証が必要。
- **ブランチ保護:** force push と履歴の書き換えはブロックされています。
- **シークレット管理:** API トークンは暗号化された GitHub Secrets として保存され、コードには含まれません。

## アセットの整合性
CloudCDN が配信するすべてのアセットは、署名付き Git コミットまでさかのぼれます。これにより以下が提供されます:
- **来歴:** すべてのファイル変更は検証済みの貢献者にリンクされています。
- **監査証跡:** 署名付きコミット検証付きの完全な Git 履歴。
- **改ざん検知:** 不正な変更があれば署名チェーンが破壊されます。

## 利用規範
CloudCDN は静的アセット配信専用です。禁止される使用には次のものが含まれます:
- マルウェアやフィッシングコンテンツのホスティング。
- 動画ストリーミングや大容量ファイル(25 MB 超)の配信。
- アセット内に個人データ、認証情報、機密情報を保存すること。
- 他のサービスの利用規約を回避するためにサービスを使用すること。

違反があった場合は 24 時間前の通知でアカウントが停止されます(違法コンテンツについては即時削除されます)。
