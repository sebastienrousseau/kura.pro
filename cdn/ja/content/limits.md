# CloudCDN — 制限とクォータ

## ファイル制限
| 制限 | 値 |
|------|-----|
| 最大ファイルサイズ(CDN 配信) | 25 MB |
| 最大ファイルサイズ(Git リポジトリ) | 厳密な制限なし(GitHub LFS が利用可能) |
| サポートされる画像形式 | PNG、JPEG、WebP、AVIF、SVG、ICO |
| サポートされる動画形式 | MP4(≤ 25 MB) |
| 最大ファイル名長 | 255 文字 |
| URL の大文字小文字の区別 | あり ― パスは大文字小文字を区別します |

## 帯域幅制限
| プラン | 月間帯域幅 | 超過時 |
|--------|-----------|-------|
| 無料 | 10 GB | サービスが翌月まで一時停止 |
| Pro | 100 GB | 0.05 ドル/GB |
| Enterprise | 無制限 | 該当なし |

帯域幅は、エッジからエンドユーザーに配信されたバイト数として測定されます。オリジンプル、CI/CD 転送、Concierge API 呼び出しはカウントされません。

## デプロイ制限
| 制限 | 値 |
|------|-----|
| 1 デプロイあたりの最大ファイル数 | 20,000 |
| 最大デプロイサイズ | 厳密な上限なし(増分アップロード) |
| 同時デプロイ数 | 1(重複時はキューに入る) |
| デプロイ頻度 | 制限なし(`main` への各プッシュで起動) |

## 画像変換の制限(Pro 以上)
| 制限 | Pro | Enterprise |
|------|-----|------------|
| 月間変換数 | 50,000 | 無制限 |
| 最大出力サイズ | 8192 × 8192 px | 8192 × 8192 px |
| 最大品質パラメータ | 100 | 100 |
| 最大ぼかし半径 | 250 | 250 |
| サポートされる出力形式 | auto、webp、avif、png、jpeg | auto、webp、avif、png、jpeg |

## API とレート制限
| エンドポイント | 制限 |
|---------------|-----|
| アセット配信 | 無制限(Cloudflare エッジ) |
| manifest.json | 無制限(5 分間のエッジキャッシュ) |
| Concierge チャット API | 1,000 リクエスト/月(全プラン) |
| キャッシュパージ API(Pro 以上) | 1,000 パージ/日 |

## カスタムドメイン
| プラン | カスタムドメイン |
|--------|----------------|
| 無料 | 0 |
| Pro | 最大 5 |
| Enterprise | 無制限(ワイルドカードを含む) |

## ストレージ
ストレージクォータはありません — リポジトリに無制限のファイルをプッシュできます。実用的な制限は GitHub のリポジトリサイズ推奨値(理想的には 5 GB 未満、より大きなリポジトリには GitHub LFS)に従います。

## 自動変換
| 制限 | 値 |
|------|-----|
| アップロードあたり生成される形式 | 2(WebP + AVIF) |
| 最大同時変換数(CI) | GitHub Actions ランナーに依存(2 CPU コア) |
| 変換タイムアウト | 6 時間(GitHub Actions の制限) |

## Concierge AI
| 制限 | 値 |
|------|-----|
| 月間クエリ数 | 1,000 |
| セッションごとのクエリ(クライアント側) | 100 |
| 会話履歴 | 現在のセッションのみ(永続化されない) |
| ナレッジベースのサイズ | 5 個のコンテンツドキュメント、約 30 チャンク |
| 応答の最大トークン数 | 512 |

## 制限に達した場合の挙動
- **帯域幅超過(無料):** その月の残り期間、アセット配信が停止します。80% 使用時にメール警告。
- **帯域幅超過(Pro):** 超過分は 0.05 ドル/GB で請求されます。サービス中断はありません。
- **Concierge 制限到達:** チャットウィジェットがその月の残り期間無効になります。
- **変換制限到達(Pro):** 変換は翌月まで元の形式を返します。
- **ファイルが大きすぎる:** 25 MB を超えるファイルは CDN から除外されますが、Git には残ります。
