# CloudCDN vs Imgix

## 한눈에 보기

| 기능 | CloudCDN | Imgix |
|------|----------|-------|
| **시작 가격** | 월 $29 (Pro) | 월 $25 (Basic) |
| **무료 플랜** | 월 10 GB 대역폭 | 없음 |
| **요금 모델** | 대역폭 (GB) | 원본 이미지 + 대역폭 |
| **워크플로** | Git 네이티브 (git push) | 대시보드/API/SDK |
| **자동 최적화** | 푸시 시 WebP + AVIF + JXL | 즉시 처리만 가능 |
| **변환 매개변수** | 9개 (resize, format, blur, sharpen, gravity, fit, quality) | 400개 이상 |
| **AI 기능** | 시맨틱 검색 + RAG 컨시어지 + MCP 서버 | 얼굴 감지, 자동 자르기 |
| **서명된 커밋** | 필수 (공급망 무결성) | 해당 없음 |
| **엣지 위치** | 300개 이상 (Cloudflare) | Imgix CDN (~50개 PoP) |
| **TTFB** | 중앙값 50ms 미만 (NA/EU) | 일반적으로 60-100ms |
| **JPEG XL** | 지원 (푸시 시 자동 생성) | 지원 |
| **에셋 출처** | 암호화 (서명된 Git 커밋) | 없음 |
| **MCP 서버** | 내장 | 제공되지 않음 |

## CloudCDN을 선택해야 할 때

- **무료 플랜**을 원할 때 (Imgix는 무료 플랜이 없음)
- SDK 없이 **Git 네이티브 워크플로**를 원할 때
- 컴플라이언스를 위한 **암호학적 에셋 출처**가 필요할 때
- 주로 정적 브랜드 에셋을 제공하며 400개의 변환 매개변수가 필요하지 않을 때
- MCP를 통한 **AI 에이전트 통합**을 원할 때

## Imgix를 선택해야 할 때

- 복잡한 이미지 파이프라인을 위한 **400개 이상의 변환 매개변수**가 필요할 때
- 사용자가 업로드한 콘텐츠의 **실시간 렌더링**이 필요할 때
- **URL 패턴 기반 퍼지**(정규식 기반)가 필요할 때
- 이미지가 S3/GCS에 저장되어 있고 처리 프록시가 필요할 때

## 비용 비교

500개의 소스 이미지와 월 50 GB 전송이 있는 사이트의 경우:

- **CloudCDN Pro**: 월 $29
- **Imgix Basic**: 월 $25 + 대역폭 초과 요금

1,000개 이상의 소스 이미지의 경우:

- **CloudCDN Pro**: 월 $29 (소스 이미지 무제한)
- **Imgix Growth**: 월 $95 (1,500개 원본 이미지 포함)
