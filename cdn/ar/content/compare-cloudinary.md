# CloudCDN مقابل Cloudinary

## نظرة سريعة

| الميزة | CloudCDN | Cloudinary |
|--------|----------|------------|
| **سعر البداية** | 29 دولارًا/شهر (Pro) | 89 دولارًا/شهر (Plus) |
| **الباقة المجانية** | 10 جيجابايت/شهر من النطاق الترددي | 25 رصيدًا/شهر (~5 آلاف تحويل) |
| **نموذج الفوترة** | النطاق الترددي (جيجابايت) | قائم على الأرصدة (معقد) |
| **سير العمل** | Git أصلي (git push) | رفع عبر لوحة التحكم/SDK/API |
| **التحسين التلقائي** | WebP + AVIF + JXL عند الدفع | فوري فقط |
| **ميزات الذكاء الاصطناعي** | بحث دلالي + كونسيرج RAG | إزالة الخلفية بالذكاء الاصطناعي، التعبئة التوليدية |
| **الالتزامات الموقعة** | مطلوبة (سلامة سلسلة التوريد) | غير قابل للتطبيق |
| **مواقع الحافة** | أكثر من 300 (Cloudflare) | ~60 نقطة تواجد للـ CDN |
| **TTFB** | أقل من 50 مللي ثانية وسطيًا (NA/EU) | عادة 80-120 مللي ثانية |
| **النطاقات المخصصة** | 5 (Pro)، غير محدودة (Enterprise) | غير محدودة (الباقات المدفوعة) |
| **SLA** | 99.9% (Pro)، 99.99% (Enterprise) | 99.9% (Business+) |
| **SSO/SAML** | Enterprise | Enterprise |
| **خادم MCP** | مدمج (تكامل وكلاء الذكاء الاصطناعي) | غير متوفر |
| **مصدر الأصول** | تشفيري (التزامات Git موقعة) | طابع زمني للرفع فقط |

## متى تختار CloudCDN

- تريد **سير عمل Git أصلي** — بدون لوحة تحكم، بدون SDK، فقط `git push`
- تحتاج إلى **مصدر تشفيري** لكل أصل (الامتثال، مسار التدقيق)
- تريد **فوترة مباشرة للنطاق الترددي** (بدون حسابات أرصدة)
- تقدم **أصولًا ثابتة** (شعارات، أيقونات، لافتات، صور) وتريد TTFB أقل من 50 مللي ثانية
- تريد **بحث الأصول المدعوم بالذكاء الاصطناعي** وخادم **MCP** لسير العمل المدفوع بالوكلاء

## متى تختار Cloudinary

- تحتاج إلى **ذكاء اصطناعي متقدم للصور** (إزالة الخلفية، التعبئة التوليدية، الاقتصاص التلقائي)
- تحتاج إلى **تحويل ترميز الفيديو** مع البث التكيفي
- تحتاج إلى **أكثر من 400 معامل تحويل** لخطوط أنابيب الصور المعقدة
- لديك تكامل Cloudinary موجود مع SDKs

## مقارنة التكلفة

لموقع يقدم 50 جيجابايت/شهر من الأصول الثابتة:

- **CloudCDN Pro**: 29 دولارًا/شهر ثابت
- **Cloudinary Plus**: 89 دولارًا/شهر + احتمال تجاوز الأرصدة

لـ 200 جيجابايت/شهر:

- **CloudCDN Pro**: 29 دولارًا + (100 جيجابايت تجاوز × 0.05 دولار) = 34 دولارًا/شهر
- **Cloudinary Advanced**: 224 دولارًا/شهر
