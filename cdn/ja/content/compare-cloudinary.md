# CloudCDN と Cloudinary の比較

## 一目でわかる違い

| 機能 | CloudCDN | Cloudinary |
|------|----------|------------|
| **入門価格** | 29 ドル/月(Pro) | 89 ドル/月(Plus) |
| **無料枠** | 10 GB/月の帯域幅 | 25 クレジット/月(約 5,000 変換) |
| **課金モデル** | 帯域幅(GB) | クレジット制(複雑) |
| **ワークフロー** | Git ネイティブ(git push) | ダッシュボード/SDK/API アップロード |
| **自動最適化** | プッシュ時に WebP + AVIF + JXL | オンザフライのみ |
| **AI 機能** | セマンティック検索 + RAG コンシェルジュ | AI 背景除去、生成塗りつぶし |
| **署名付きコミット** | 必須(サプライチェーン整合性) | 該当なし |
| **エッジロケーション** | 300 以上(Cloudflare) | 約 60 CDN PoP |
| **TTFB** | 中央値 50ms 未満(NA/EU) | 通常 80-120ms |
| **カスタムドメイン** | 5(Pro)、無制限(Enterprise) | 無制限(有料プラン) |
| **SLA** | 99.9%(Pro)、99.99%(Enterprise) | 99.9%(Business 以上) |
| **SSO/SAML** | Enterprise | Enterprise |
| **MCP サーバー** | 内蔵(AI エージェント統合) | 利用不可 |
| **アセットの来歴** | 暗号学的(署名付き Git コミット) | アップロード時刻のみ |

## CloudCDN を選ぶべき場合

- **Git ネイティブなワークフロー**が欲しい — ダッシュボードも SDK も不要、`git push` だけ
- 各アセットに**暗号学的な来歴**が必要(コンプライアンス、監査証跡)
- **シンプルな帯域幅課金**が欲しい(クレジット計算なし)
- **静的アセット**(ロゴ、アイコン、バナー、画像)を配信し、50ms 未満の TTFB を求める
- エージェント駆動のワークフロー向けに **AI 検索**と **MCP サーバー**が欲しい

## Cloudinary を選ぶべき場合

- **高度な画像 AI**(背景除去、生成塗りつぶし、自動トリミング)が必要
- **アダプティブストリーミング**による**動画トランスコード**が必要
- 複雑な画像パイプライン向けに **400 以上の変換パラメータ**が必要
- すでに SDK を使った Cloudinary 連携が存在する

## コスト比較

月 50 GB の静的アセットを配信するサイトの場合:

- **CloudCDN Pro**: 29 ドル/月の定額
- **Cloudinary Plus**: 89 ドル/月 + クレジットの追加料金が発生する可能性

月 200 GB の場合:

- **CloudCDN Pro**: 29 ドル + (100 GB 超過 × 0.05 ドル) = 34 ドル/月
- **Cloudinary Advanced**: 224 ドル/月
