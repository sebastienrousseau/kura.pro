# CloudCDN 与 Cloudinary 对比

## 速览

| 功能 | CloudCDN | Cloudinary |
|------|----------|------------|
| **起步价格** | $29/月（Pro） | $89/月（Plus） |
| **免费套餐** | 10 GB/月带宽 | 25 个积分/月（约 5K 次转换） |
| **计费模式** | 带宽（GB） | 基于积分（复杂） |
| **工作流** | Git 原生（git push） | 控制台/SDK/API 上传 |
| **自动优化** | 推送时生成 WebP + AVIF + JXL | 仅即时处理 |
| **AI 功能** | 语义搜索 + RAG 礼宾 | AI 背景去除、生成式填充 |
| **签名提交** | 必须（供应链完整性） | 不适用 |
| **边缘节点** | 300+（Cloudflare） | 约 60 个 CDN PoP |
| **TTFB** | 中位数 <50ms（北美/欧洲） | 通常 ~80-120ms |
| **自定义域名** | 5 个（Pro），无限（Enterprise） | 无限（付费套餐） |
| **SLA** | 99.9%（Pro），99.99%（Enterprise） | 99.9%（Business+） |
| **SSO/SAML** | Enterprise | Enterprise |
| **MCP 服务器** | 内置（AI 代理集成） | 不可用 |
| **资源溯源** | 加密（签名 Git 提交） | 仅上传时间戳 |

## 何时选择 CloudCDN

- 你想要 **Git 原生工作流** — 没有控制台、没有 SDK，只需 `git push`
- 你需要每个资源的 **加密溯源**（合规、审计追踪）
- 你想要 **简单的带宽计费**（无需积分计算）
- 你提供 **静态资源**（徽标、图标、横幅、图像），并希望 TTFB 低于 50ms
- 你想要 **AI 驱动的资源搜索** 和用于代理驱动工作流的 **MCP 服务器**

## 何时选择 Cloudinary

- 你需要 **高级图像 AI**（背景去除、生成式填充、自动裁剪）
- 你需要支持自适应流媒体的 **视频转码**
- 你需要 **400+ 转换参数** 用于复杂的图像管道
- 你已经有使用 SDK 的 Cloudinary 集成

## 成本对比

对于每月提供 50 GB 静态资源的网站：

- **CloudCDN Pro**：$29/月固定
- **Cloudinary Plus**：$89/月 + 潜在的积分超额费用

对于每月 200 GB：

- **CloudCDN Pro**：$29 + (100 GB 超额 × $0.05) = $34/月
- **Cloudinary Advanced**：$224/月
