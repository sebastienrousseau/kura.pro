## 무료 등급 (오픈소스)
- **대상:** 개인 프로젝트, 오픈소스 소프트웨어, 취미 사용자.
- **비용:** 월 $0 — 영구 무료, 신용 카드 불필요.
- **대역폭:** 월 10 GB. 초과 시 CDN을 통한 에셋 제공이 중단됩니다 (파일은 git에 남아 있음).
- **에셋:** 무제한 정적 파일 (PNG, WebP, AVIF, SVG, ICO).
- **자동 최적화:** 푸시된 모든 PNG/JPEG는 WebP (품질 80) 및 AVIF (품질 65)로 자동 변환됩니다.
- **전송:** Cloudflare의 300+ 엣지 PoP에서 100ms 미만 TTFB. 불변 캐싱 (1년 max-age).
- **제한사항:** 사용자 정의 도메인 없음, 이미지 변환 API 없음, 분석 대시보드 없음.

## Pro 등급
- **대상:** 상업 프로젝트, 스타트업, 트래픽이 많은 웹사이트.
- **비용:** 월 $29 (또는 연 $278 — 20% 절약).
- **대역폭:** 월 100 GB 포함. 초과: 주기 종료 시 GB당 $0.05 청구.
- **이미지 변환 API:** URL 매개변수를 통해 즉시 크기 조정, 자르기, 변환:
  ```
  https://cloudcdn.pro/프로젝트/image.png?w=800&h=600&fit=cover&format=auto&q=80
  ```
- **형식 협상:** 브라우저 `Accept` 헤더를 기반으로 한 자동 AVIF/WebP 제공.
- **사용자 정의 도메인:** 자동 SSL 프로비저닝과 함께 최대 5개 도메인.
- **우선 지원:** 24시간 응답 보장이 있는 이메일 지원.
- **SLA:** 99.9% 가동 시간. 위반 시 임계값 미만 0.1%당 10% 서비스 크레딧.
- **분석:** 대역폭, 요청 수, 캐시 적중률, 형식 분포.
- **신용 카드 없이 전체 액세스가 가능한 14일 무료 평가판.**

## Enterprise 등급
- **대상:** 글로벌 플랫폼, 미디어 회사, 대규모 전자상거래.
- **비용:** 맞춤형 가격 — sales@cloudcdn.pro로 문의하세요.
- **대역폭:** 전용 엣지 할당과 함께 무제한.
- **전체 API 제품군:** 모든 Pro 변환과 일괄 처리, 웹훅 알림, 프로그래매틱 업로드.
- **사용자 정의 도메인:** 와일드카드 SSL과 함께 무제한.
- **전담 지원:** 지정된 계정 매니저, 전용 Slack 채널, 1시간 응답 SLA.
- **SLA:** 99.99% 가동 시간. 금전적 크레딧: <99.99%에 대해 25%, <99.9%에 대해 50%, <99.0%에 대해 100%.
- **보안:** SSO/SAML, 12개월 보존 감사 로그, IP 허용 목록.
- **분석:** 에셋별, 지역별, 형식별 분석이 있는 실시간 대시보드.
- **컴플라이언스:** GDPR DPA 사용 가능, 데이터 거주 옵션 (EU/US).

## 청구
- 기본적으로 월별 청구. 연간 청구로 20% 절약.
- 무료 등급: 신용 카드 없음, 평가판 만료 없음.
- Pro 평가판: 14일, 전체 기능 액세스, 신용 카드 없음.
- 초과: 주기 종료 시 청구되며, 주기 중간에 서비스 중단 없음.
- 언제든지 취소. 장기 계약 없음. 데이터는 취소 후 30일 동안 액세스 가능합니다.

## 공정 사용 정책
CloudCDN은 정적 에셋(이미지, 아이콘, 폰트, 문서) 제공을 위해 설계되었습니다. 비디오 스트리밍(>25 MB 파일), 애플리케이션 바이너리 배포 또는 파일 호스팅을 위한 것이 아닙니다. 공정 사용을 초과하는 계정은 적절한 플랜에 대해 논의하기 위해 연락드릴 것입니다. 예고 없이 절대로 정지하지 않습니다.
