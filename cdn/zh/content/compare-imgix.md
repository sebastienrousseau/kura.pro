# CloudCDN 与 Imgix 对比

## 速览

| 功能 | CloudCDN | Imgix |
|------|----------|-------|
| **起步价格** | $29/月（Pro） | $25/月（Basic） |
| **免费套餐** | 10 GB/月带宽 | 无 |
| **计费模式** | 带宽（GB） | 源图像 + 带宽 |
| **工作流** | Git 原生（git push） | 控制台/API/SDK |
| **自动优化** | 推送时生成 WebP + AVIF + JXL | 仅即时处理 |
| **转换参数** | 9 个（resize、format、blur、sharpen、gravity、fit、quality） | 400+ |
| **AI 功能** | 语义搜索 + RAG 礼宾 + MCP 服务器 | 人脸检测、自动裁剪 |
| **签名提交** | 必须（供应链完整性） | 不适用 |
| **边缘节点** | 300+（Cloudflare） | Imgix CDN（约 50 个 PoP） |
| **TTFB** | 中位数 <50ms（北美/欧洲） | 通常 ~60-100ms |
| **JPEG XL** | 支持（推送时自动生成） | 支持 |
| **资源溯源** | 加密（签名 Git 提交） | 无 |
| **MCP 服务器** | 内置 | 不可用 |

## 何时选择 CloudCDN

- 你想要 **免费套餐**（Imgix 没有）
- 你想要 **Git 原生工作流**，无需 SDK
- 你需要用于合规的 **加密资源溯源**
- 你主要提供静态品牌资源，不需要 400 个转换参数
- 你想要通过 MCP 进行 **AI 代理集成**

## 何时选择 Imgix

- 你需要 **400+ 转换参数** 用于复杂的图像管道
- 你需要用户上传内容的 **实时渲染**
- 你需要 **按 URL 模式清除缓存**（基于正则表达式）
- 你的图像存储在 S3/GCS 中，需要处理代理

## 成本对比

对于具有 500 张源图像和每月 50 GB 交付的网站：

- **CloudCDN Pro**：$29/月
- **Imgix Basic**：$25/月 + 带宽超额费用

对于 1,000+ 张源图像：

- **CloudCDN Pro**：$29/月（源图像无限制）
- **Imgix Growth**：$95/月（包含 1,500 张源图像）
