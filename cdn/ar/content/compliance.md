# CloudCDN — الامتثال والخصوصية

## معالجة البيانات

### ما هي البيانات التي يتعامل معها CloudCDN؟
يقدم CloudCDN ملفات ثابتة (صور، أيقونات، خطوط). لا يعالج أو يخزن أو ينقل بيانات شخصية للمستخدمين. تدفق البيانات الوحيد هو:
1. يطلب المتصفح عنوان URL لملف ثابت.
2. تقدم حافة Cloudflare الملف المخزن مؤقتًا.
3. يتم إنشاء سجلات HTTP قياسية (IP، الطابع الزمني، URL، user-agent).

### أين يتم تخزين البيانات؟
- **الملفات المصدرية:** مستودع GitHub (مستضاف في الولايات المتحدة من قبل GitHub/Microsoft).
- **التخزين المؤقت للحافة:** أكثر من 300 نقطة تواجد عالمية لـ Cloudflare. يتم توزيع النسخ المخزنة مؤقتًا في جميع أنحاء العالم لتحسين الأداء.
- **بيانات كونسيرج الذكاء الاصطناعي:** لا يتم تخزين المحادثات. يستخدم عنصر الدردشة حالة الجلسة في الذاكرة فقط — لا يوجد تسجيل من جانب الخادم لاستفسارات المستخدم.
- **عدادات تحديد المعدل:** مخزنة في Cloudflare Workers KV (أعداد مجمعة فقط، لا توجد بيانات شخصية).

## الامتثال للائحة GDPR

### الحالة
يمتثل CloudCDN للائحة GDPR. نستخدم Cloudflare كمزود للبنية التحتية، والذي يحافظ على الامتثال للائحة GDPR من خلال:
- شهادة EU-US Data Privacy Framework.
- البنود التعاقدية القياسية (SCCs) لنقل البيانات الدولي.
- اتفاقيات معالجة البيانات متاحة عند الطلب.

### تقليل البيانات
- لا يضع CloudCDN أي ملفات تعريف ارتباط.
- لا يوجد تتبع للمستخدمين أو بكسلات تحليلية.
- لا يتم جمع أو تخزين أو معالجة أي بيانات شخصية.
- يتم التعامل مع سجلات وصول HTTP بواسطة Cloudflare وفقًا لسياسة الخصوصية الخاصة بهم.

### حقوق أصحاب البيانات
نظرًا لأن CloudCDN لا يجمع بيانات شخصية، فلا توجد بيانات شخصية للوصول إليها أو تصحيحها أو حذفها. إذا كنت تعتقد أن بياناتك الشخصية قد تم تضمينها عن غير قصد في أحد الأصول (على سبيل المثال، صورة)، فاتصل بـ support@cloudcdn.pro لإزالتها.

### DPA (اتفاقية معالجة البيانات)
يمكن لعملاء Enterprise طلب DPA رسمي. اتصل بـ sales@cloudcdn.pro.

## CCPA / CPRA (كاليفورنيا)
لا يبيع CloudCDN أو يشارك أو يستخدم المعلومات الشخصية للإعلانات الموجهة. لا يلزم وجود آلية إلغاء الاشتراك حيث لا يتم جمع أي بيانات شخصية.

## SOC 2 / ISO 27001
يستفيد CloudCDN من البنية التحتية لـ Cloudflare، والتي تحافظ على:
- شهادة SOC 2 Type II.
- شهادة ISO 27001.
- امتثال PCI DSS Level 1.
تغطي هذه الشهادات البنية التحتية لتسليم الحافة التي يستخدمها CloudCDN.

## التدابير الأمنية
- **التشفير أثناء النقل:** TLS 1.3 على جميع الاتصالات.
- **حماية DDoS:** التخفيف التلقائي لـ DDoS من Cloudflare على جميع الباقات.
- **WAF:** جدار حماية تطبيقات الويب من Cloudflare نشط على جميع نقاط النهاية.
- **التخفيف من البوتات:** تحمي إدارة Bot من Cloudflare من الكشط وسوء الاستخدام.
- **الالتزامات الموقعة:** جميع تغييرات الأصول تتطلب التحقق التشفيري.
- **حماية الفروع:** الدفع القسري وإعادة كتابة السجل محظوران.
- **إدارة الأسرار:** يتم تخزين رموز API كأسرار GitHub مشفرة، وليس في الكود أبدًا.

## سلامة الأصول
كل أصل يقدمه CloudCDN يمكن تتبعه إلى التزام Git موقع. يوفر هذا:
- **المصدر:** كل تغيير في الملف مرتبط بمساهم تم التحقق منه.
- **مسار التدقيق:** سجل Git كامل مع التحقق من الالتزامات الموقعة.
- **اكتشاف التلاعب:** أي تعديل غير مصرح به يكسر سلسلة التوقيع.

## الاستخدام المقبول
CloudCDN مخصص لتسليم الأصول الثابتة فقط. تشمل الاستخدامات المحظورة:
- استضافة البرامج الضارة أو محتوى التصيد الاحتيالي.
- بث الفيديو أو توزيع الملفات الكبيرة (أكثر من 25 ميجابايت).
- تخزين البيانات الشخصية أو بيانات الاعتماد أو المعلومات الحساسة في الأصول.
- استخدام الخدمة للالتفاف على شروط الخدمات الأخرى.

تؤدي الانتهاكات إلى تعليق الحساب مع إشعار مدته 24 ساعة (باستثناء المحتوى غير القانوني، والذي تتم إزالته على الفور).
