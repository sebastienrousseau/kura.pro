# CloudCDN مقابل Imgix

## نظرة سريعة

| الميزة | CloudCDN | Imgix |
|--------|----------|-------|
| **سعر البداية** | 29 دولارًا/شهر (Pro) | 25 دولارًا/شهر (Basic) |
| **الباقة المجانية** | 10 جيجابايت/شهر من النطاق الترددي | لا يوجد |
| **نموذج الفوترة** | النطاق الترددي (جيجابايت) | الصور الأصلية + النطاق الترددي |
| **سير العمل** | Git أصلي (git push) | لوحة تحكم/API/SDK |
| **التحسين التلقائي** | WebP + AVIF + JXL عند الدفع | فوري فقط |
| **معاملات التحويل** | 9 (resize، format، blur، sharpen، gravity، fit، quality) | أكثر من 400 |
| **ميزات الذكاء الاصطناعي** | بحث دلالي + كونسيرج RAG + خادم MCP | اكتشاف الوجوه، الاقتصاص التلقائي |
| **الالتزامات الموقعة** | مطلوبة (سلامة سلسلة التوريد) | غير قابل للتطبيق |
| **مواقع الحافة** | أكثر من 300 (Cloudflare) | Imgix CDN (~50 نقطة تواجد) |
| **TTFB** | أقل من 50 مللي ثانية وسطيًا (NA/EU) | عادة 60-100 مللي ثانية |
| **JPEG XL** | مدعوم (يُولَّد تلقائيًا عند الدفع) | مدعوم |
| **مصدر الأصول** | تشفيري (التزامات Git موقعة) | لا يوجد |
| **خادم MCP** | مدمج | غير متوفر |

## متى تختار CloudCDN

- تريد **باقة مجانية** (لا تملك Imgix واحدة)
- تريد **سير عمل Git أصلي** بدون أي SDKs
- تحتاج إلى **مصدر تشفيري للأصول** لأغراض الامتثال
- تقدم في الغالب أصولًا ثابتة للعلامة التجارية ولا تحتاج إلى 400 معامل تحويل
- تريد **تكامل وكلاء الذكاء الاصطناعي** عبر MCP

## متى تختار Imgix

- تحتاج إلى **أكثر من 400 معامل تحويل** لخطوط أنابيب الصور المعقدة
- تحتاج إلى **العرض الفوري** للمحتوى المرفوع من المستخدمين
- تحتاج إلى **مسح حسب نمط URL** (قائم على التعابير النمطية)
- صورك مخزنة في S3/GCS وتحتاج إلى وكيل معالجة

## مقارنة التكلفة

لموقع به 500 صورة مصدرية و50 جيجابايت/شهر من التسليم:

- **CloudCDN Pro**: 29 دولارًا/شهر
- **Imgix Basic**: 25 دولارًا/شهر + تجاوز النطاق الترددي

لأكثر من 1000 صورة مصدرية:

- **CloudCDN Pro**: 29 دولارًا/شهر (صور مصدرية غير محدودة)
- **Imgix Growth**: 95 دولارًا/شهر (تشمل 1500 صورة أصلية)
