# CloudCDN と Bunny CDN の比較

## 一目でわかる違い

| 機能 | CloudCDN | Bunny CDN |
|------|----------|-----------|
| **入門価格** | 29 ドル/月(Pro) | 9.50 ドル/月の定額 |
| **無料枠** | 10 GB/月の帯域幅 | 14 日間の試用のみ |
| **課金モデル** | 帯域幅のティア制 | GB あたり(ボリューム価格 0.01 ドル/GB から) |
| **ワークフロー** | Git ネイティブ(git push) | ダッシュボード/FTP/API |
| **自動最適化** | プッシュ時に WebP + AVIF + JXL | Bunny Optimizer(別アドオン) |
| **ストレージ API** | Bunny.net 互換の JSON スキーマ | ネイティブ |
| **AI 機能** | セマンティック検索 + RAG コンシェルジュ + MCP サーバー | なし |
| **署名付きコミット** | 必須(サプライチェーン整合性) | 該当なし |
| **エッジロケーション** | 300 以上(Cloudflare) | 114 PoP |
| **TTFB** | 中央値 50ms 未満(NA/EU) | 通常 40-80ms |
| **エッジコンピュート** | Cloudflare Workers(完全な JS ランタイム) | Bunny Script(制限あり) |
| **Perma-Cache** | 1 年間の不変ヘッダー | Perma-Cache 機能 |
| **JPEG XL** | サポート | Optimizer では利用不可 |
| **アセットの来歴** | 暗号学的(署名付き Git コミット) | なし |
| **MCP サーバー** | 内蔵 | 利用不可 |

## CloudCDN が Bunny 互換 API を使う理由

CloudCDN のストレージ API は Bunny.net 互換の JSON スキーマを返します(Guid、StorageZoneName、Path、ObjectName など)。これは Bunny 用に作られた移行ツールやスクリプトが、CloudCDN でもそのまま動作することを意味します。

## CloudCDN を選ぶべき場合

- **Git ネイティブなワークフロー**を使いたい(FTP もダッシュボードアップロードも不要)
- **AI 駆動の検索**と**エージェント統合**(MCP)が必要
- 各アセットに対して**暗号学的な来歴**が必要
- WebP と AVIF に加えて、**JPEG XL の自動生成**が欲しい
- **永続的な無料枠**が欲しい(Bunny の試用は期限切れになる)

## Bunny CDN を選ぶべき場合

- **可能な限り低い GB 単価**が必要(地域によっては 0.01 ドル/GB)
- ストレージへの **FTP/SFTP アクセス**が必要
- 動画ホスティングのための **Bunny Stream** が必要
- 帯域幅が非常に多く(月 100 TB 以上)、ボリュームディスカウントが必要
- カスタムルール付きの **DDoS 保護**をアドオンとして必要

## コスト比較

月 50 GB を配信するサイトの場合:

- **CloudCDN Pro**: 29 ドル/月
- **Bunny CDN**: 約 2.50 ドル/月(EU/US で 0.05 ドル/GB)+ 0.50 ドル/月のストレージ

月 500 GB の場合:

- **CloudCDN Pro**: 29 ドル + (400 GB × 0.05 ドル) = 49 ドル/月
- **Bunny CDN**: 約 25 ドル/月(EU/US)+ ストレージ

純粋な帯域幅コストでは Bunny に軍配が上がります。CloudCDN はワークフロー、AI 機能、来歴、無料枠で勝ります。
