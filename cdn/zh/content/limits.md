# CloudCDN — 限制和配额

## 文件限制
| 限制 | 值 |
|------|-----|
| 最大文件大小（CDN 交付） | 25 MB |
| 最大文件大小（Git 仓库） | 无硬限制（可使用 GitHub LFS） |
| 支持的图像格式 | PNG、JPEG、WebP、AVIF、SVG、ICO |
| 支持的视频格式 | MP4（≤25 MB） |
| 最大文件名长度 | 255 个字符 |
| URL 区分大小写 | 是 — 路径区分大小写 |

## 带宽限制
| 套餐 | 月度带宽 | 超额 |
|------|---------|------|
| Free | 10 GB | 服务暂停直到下个月 |
| Pro | 100 GB | $0.05/GB |
| Enterprise | 无限制 | N/A |

带宽以从边缘交付给最终用户的字节数衡量。源拉取、CI/CD 传输和 Concierge API 调用不计入。

## 部署限制
| 限制 | 值 |
|------|-----|
| 每次部署的最大文件数 | 20,000 |
| 最大部署大小 | 无硬上限（增量上传） |
| 并发部署 | 1（重叠时排队） |
| 部署频率 | 无限制（每次推送到 main 时触发） |

## 图像转换限制（Pro+）
| 限制 | Pro | Enterprise |
|------|-----|-----------|
| 每月转换次数 | 50,000 | 无限制 |
| 最大输出尺寸 | 8192 x 8192 px | 8192 x 8192 px |
| 最大质量参数 | 100 | 100 |
| 最大模糊半径 | 250 | 250 |
| 支持的输出格式 | auto、webp、avif、png、jpeg | auto、webp、avif、png、jpeg |

## API 和速率限制
| 端点 | 限制 |
|------|------|
| 资源交付 | 无限制（Cloudflare 边缘） |
| manifest.json | 无限制（5 分钟边缘缓存） |
| Concierge 聊天 API | 每月 1,000 次查询（所有套餐） |
| 缓存清除 API（Pro+） | 每天 1,000 次清除 |

## 自定义域名
| 套餐 | 自定义域名 |
|------|-----------|
| Free | 0 |
| Pro | 最多 5 个 |
| Enterprise | 无限制（包括通配符） |

## 存储
没有存储配额 — 你可以向仓库推送无限的文件。实际限制是 GitHub 的仓库大小建议（理想情况下小于 5 GB，对于更大的仓库使用 GitHub LFS）。

## 自动转换
| 限制 | 值 |
|------|-----|
| 每次上传生成的格式 | 2（WebP + AVIF） |
| 最大并发转换数（CI） | 基于 GitHub Actions 运行器（2 个 CPU 核心） |
| 转换超时 | 6 小时（GitHub Actions 限制） |

## Concierge AI
| 限制 | 值 |
|------|-----|
| 每月查询次数 | 1,000 |
| 每会话查询次数（客户端） | 100 |
| 对话历史 | 仅当前会话（不持久化） |
| 知识库大小 | 5 个内容文档，约 30 个块 |
| 响应最大令牌数 | 512 |

## 达到限制时会发生什么
- **带宽超出（Free）：** 资源停止提供直到下个月。80% 时通过电子邮件警告。
- **带宽超出（Pro）：** 超额按 $0.05/GB 计费。无服务中断。
- **Concierge 限制达到：** 聊天小部件在该月内禁用。
- **转换限制达到（Pro）：** 转换返回原始格式直到下个月。
- **文件过大：** 大于 25 MB 的文件被排除在 CDN 之外，但保留在 Git 中。
