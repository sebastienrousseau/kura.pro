# CloudCDN — 제한 및 할당량

## 파일 제한
| 제한 | 값 |
|------|-----|
| 최대 파일 크기 (CDN 전송) | 25 MB |
| 최대 파일 크기 (Git 리포지토리) | 엄격한 제한 없음 (GitHub LFS 사용 가능) |
| 지원되는 이미지 형식 | PNG, JPEG, WebP, AVIF, SVG, ICO |
| 지원되는 비디오 형식 | MP4 (≤25 MB) |
| 최대 파일명 길이 | 255자 |
| URL 대소문자 구분 | 예 — 경로는 대소문자를 구분합니다 |

## 대역폭 제한
| 등급 | 월 대역폭 | 초과 |
|------|-----------|------|
| Free | 10 GB | 다음 달까지 서비스 일시 중지 |
| Pro | 100 GB | GB당 $0.05 |
| Enterprise | 무제한 | N/A |

대역폭은 엣지에서 최종 사용자에게 전달된 바이트로 측정됩니다. 원본 풀, CI/CD 전송 및 Concierge API 호출은 계산되지 않습니다.

## 배포 제한
| 제한 | 값 |
|------|-----|
| 배포당 최대 파일 수 | 20,000 |
| 최대 배포 크기 | 엄격한 제한 없음 (증분 업로드) |
| 동시 배포 | 1 (겹치는 경우 대기열에 추가됨) |
| 배포 빈도 | 제한 없음 (main으로의 모든 푸시 시 트리거됨) |

## 이미지 변환 제한 (Pro+)
| 제한 | Pro | Enterprise |
|------|-----|-----------|
| 월 변환 횟수 | 50,000 | 무제한 |
| 최대 출력 치수 | 8192 x 8192 px | 8192 x 8192 px |
| 최대 품질 매개변수 | 100 | 100 |
| 최대 블러 반경 | 250 | 250 |
| 지원되는 출력 형식 | auto, webp, avif, png, jpeg | auto, webp, avif, png, jpeg |

## API 및 속도 제한
| 엔드포인트 | 제한 |
|------------|------|
| 에셋 전송 | 무제한 (Cloudflare 엣지) |
| manifest.json | 무제한 (5분 엣지 캐시) |
| Concierge 챗 API | 월 1,000 쿼리 (모든 등급) |
| 캐시 퍼지 API (Pro+) | 일 1,000 퍼지 |

## 사용자 정의 도메인
| 등급 | 사용자 정의 도메인 |
|------|--------------------|
| Free | 0 |
| Pro | 최대 5개 |
| Enterprise | 무제한 (와일드카드 포함) |

## 스토리지
스토리지 할당량이 없습니다 — 리포지토리에 무제한으로 파일을 푸시할 수 있습니다. 실용적인 제한은 GitHub의 리포지토리 크기 권장 사항(이상적으로 5 GB 미만, 더 큰 리포지토리의 경우 GitHub LFS 사용)입니다.

## 자동 변환
| 제한 | 값 |
|------|-----|
| 업로드당 생성되는 형식 | 2 (WebP + AVIF) |
| 최대 동시 변환 (CI) | GitHub Actions 러너 기준 (2 CPU 코어) |
| 변환 타임아웃 | 6시간 (GitHub Actions 제한) |

## Concierge AI
| 제한 | 값 |
|------|-----|
| 월 쿼리 | 1,000 |
| 세션 쿼리 (클라이언트 측) | 100 |
| 대화 히스토리 | 현재 세션만 (지속되지 않음) |
| 지식 베이스 크기 | 5개 콘텐츠 문서, ~30개 청크 |
| 응답 최대 토큰 | 512 |

## 제한에 도달하면 어떻게 됩니까
- **대역폭 초과 (Free):** 다음 달까지 에셋 제공이 중단됩니다. 80%에서 이메일 경고.
- **대역폭 초과 (Pro):** 초과분은 GB당 $0.05로 청구됩니다. 서비스 중단 없음.
- **Concierge 한도 도달:** 챗 위젯이 해당 월 동안 비활성화됩니다.
- **변환 한도 도달 (Pro):** 변환은 다음 달까지 원본 형식을 반환합니다.
- **파일이 너무 큼:** 25 MB 이상의 파일은 CDN에서 제외되지만 Git에는 남아 있습니다.
