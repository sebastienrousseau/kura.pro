# CloudCDN と Imgix の比較

## 一目でわかる違い

| 機能 | CloudCDN | Imgix |
|------|----------|-------|
| **入門価格** | 29 ドル/月(Pro) | 25 ドル/月(Basic) |
| **無料枠** | 10 GB/月の帯域幅 | なし |
| **課金モデル** | 帯域幅(GB) | オリジン画像 + 帯域幅 |
| **ワークフロー** | Git ネイティブ(git push) | ダッシュボード/API/SDK |
| **自動最適化** | プッシュ時に WebP + AVIF + JXL | オンザフライのみ |
| **変換パラメータ** | 9(リサイズ、形式、ぼかし、シャープ、グラビティ、フィット、品質) | 400 以上 |
| **AI 機能** | セマンティック検索 + RAG コンシェルジュ + MCP サーバー | 顔検出、自動トリミング |
| **署名付きコミット** | 必須(サプライチェーン整合性) | 該当なし |
| **エッジロケーション** | 300 以上(Cloudflare) | Imgix CDN(約 50 PoP) |
| **TTFB** | 中央値 50ms 未満(NA/EU) | 通常 60-100ms |
| **JPEG XL** | サポート(プッシュ時に自動生成) | サポート |
| **アセットの来歴** | 暗号学的(署名付き Git コミット) | なし |
| **MCP サーバー** | 内蔵 | 利用不可 |

## CloudCDN を選ぶべき場合

- **無料枠**が欲しい(Imgix にはない)
- SDK 不要の **Git ネイティブなワークフロー**が欲しい
- コンプライアンスのための**暗号学的なアセット来歴**が必要
- 主に静的なブランドアセットを配信しており、400 もの変換パラメータは不要
- MCP 経由の **AI エージェント統合**が欲しい

## Imgix を選ぶべき場合

- 複雑な画像パイプラインのために **400 以上の変換パラメータ**が必要
- ユーザーがアップロードするコンテンツの**リアルタイムレンダリング**が必要
- **URL パターンによるパージ**(正規表現ベース)が必要
- 画像が S3/GCS に保存されており、処理プロキシが必要

## コスト比較

500 個のソース画像と月 50 GB の配信を行うサイトの場合:

- **CloudCDN Pro**: 29 ドル/月
- **Imgix Basic**: 25 ドル/月 + 帯域幅の超過料金

1,000 個以上のソース画像の場合:

- **CloudCDN Pro**: 29 ドル/月(ソース画像数は無制限)
- **Imgix Growth**: 95 ドル/月(1,500 オリジン画像を含む)
