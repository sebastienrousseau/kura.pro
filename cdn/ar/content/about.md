# حول CloudCDN

## ما هو CloudCDN؟
CloudCDN هي شبكة تسليم أصول ثابتة أصلية في Git مصممة للمطورين. ادفع الصور إلى مستودع GitHub، وستُحسَّن تلقائياً وتُقدَّم عالمياً من شبكة Cloudflare الطرفية: أكثر من 300 مركز بيانات، وأكثر من 100 دولة، وزمن استجابة أقل من 100 مللي ثانية.

على عكس شبكات CDN التقليدية للصور التي تتطلب لوحات تحكم أو واجهات API للرفع أو تكاملات SDK، يستخدم CloudCDN سير العمل الذي يعرفه المطورون بالفعل: `git push`.

## كيف يعمل
1. **الدفع:** أضف الصور إلى مستودع GitHub والتزم بمفتاح موقَّع.
2. **التحسين:** يضغط GitHub Actions الصور تلقائياً وينشئ متغيرات WebP (جودة 80، أصغر بنحو 60٪) وAVIF (جودة 65، أصغر بنحو 70٪).
3. **النشر:** تُرفع الملفات المعدَّلة إلى Cloudflare Pages عبر النشر التزايدي (إزالة التكرار بناءً على تجزئة المحتوى — تُنقل فقط الملفات الجديدة أو المعدَّلة).
4. **التسليم:** تُقدَّم الأصول مع رؤوس ذاكرة تخزين مؤقت غير قابلة للتغيير (`max-age=31536000`) من أقرب موقع طرفي من بين أكثر من 300 موقع.

## مكدس التقنيات
- **الشبكة الطرفية:** Cloudflare Pages على أكثر من 300 PoP عالمي. 95٪ من سكان العالم يبعدون أقل من 50 مللي ثانية عن أحد مراكز بيانات Cloudflare.
- **صيغ الصور:** PNG (الأصل بدون خسارة)، WebP (مع خسارة، جودة 80)، AVIF (مع خسارة، جودة 65)، SVG (تمرير مباشر)، ICO (تمرير مباشر).
- **التفاوض على الصيغة:** يقدم باقة Pro الصيغة المثلى استناداً إلى رأس `Accept` للمتصفح (AVIF > WebP > الأصلي).
- **التخزين المؤقت:** تخزين مؤقت طرفي وتخزين متصفح غير قابلين للتغيير مع max-age لمدة عام واحد. إبطال ذاكرة التخزين المؤقت بتغيير اسم الملف أو المسار.
- **CI/CD:** GitHub Actions للضغط (Sharp)، وإنشاء البيان، والنشر على Cloudflare Pages (Wrangler).
- **الأمان:** التزامات موقَّعة بـ SSH Ed25519 إلزامية. حماية الفرع `main`. رموز API مشفَّرة عبر GitHub Secrets.
- **الكونسيرج بالذكاء الاصطناعي:** Cloudflare Workers AI (Llama 3.1) + Vectorize RAG للبحث الذكي في الوثائق من الصفحة الرئيسية.

## الأداء
- **TTFB:** متوسط أقل من 50 مللي ثانية في أمريكا الشمالية/أوروبا، وأقل من 100 مللي ثانية عالمياً (إصابة ذاكرة التخزين المؤقت الطرفية لـ Cloudflare).
- **نسبة إصابة ذاكرة التخزين المؤقت:** أكثر من 95٪ لأصول الإنتاج (تخزين مؤقت غير قابل للتغيير).
- **سرعة النشر:** عمليات رفع تزايدية — تُنقل فقط الملفات المعدَّلة. النشر النموذجي: 5 إلى 30 ثانية.
- **الضغط:** WebP يوفر حوالي 60٪ مقارنة بـ PNG. AVIF يوفر حوالي 70٪ مقارنة بـ PNG. يُنشأ كلاهما تلقائياً.

## تنظيم الأصول
```
project-name/
  images/
    banners/    — رسومات بتنسيق عريض (1200x630 موصى به)
    icons/      — متعددة الدقة (من 16x16 إلى 512x512، مع @2x/@3x)
    logos/      — شعارات وعلامات تجارية
    github/     — صور المعاينة الاجتماعية
    titles/     — رسومات العناوين والترويسات
  README.md     — وصف اختياري للمشروع
```

## المقاييس الرئيسية
- **أكثر من 1,400 أصل محسَّن** موزعة على 54 منطقة مستأجر.
- **مصدر واحد لكل صورة** — تُولَّد المشتقات عند الطلب عبر `/api/transform`.
- **أكثر من 300 PoP طرفي** في أكثر من 100 دولة.
- **TTFB عالمي أقل من 100 مللي ثانية** عند إصابة ذاكرة التخزين المؤقت الطرفية.
- **بدون خطوة بناء** — لا حاجة لـ `npm install` أو webpack أو أي إطار عمل.

## من يستخدم CloudCDN؟
يقدم CloudCDN الأصول لـ:
- **مشاريع البرمجيات مفتوحة المصدر:** الشعارات والبنرات والأيقونات ورسومات الوثائق.
- **أدوات المطورين:** علامات مطوري Rust وPython والذكاء الاصطناعي (rustdev وpythondev وllamadev).
- **منصات التكنولوجيا المالية:** مكتبات أصول للخدمات المصرفية والحوسبة الكمية.
- **التطبيقات الصوتية:** تصورات الموجات الصوتية ومكونات الواجهة.
- **مولدات المواقع الثابتة:** Shokunin وKaishi وأطر عمل SSG الأخرى.
