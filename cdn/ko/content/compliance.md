# CloudCDN — 컴플라이언스 및 개인정보 보호

## 데이터 처리

### CloudCDN은 어떤 데이터를 처리합니까?
CloudCDN은 정적 파일(이미지, 아이콘, 폰트)을 제공합니다. 개인 사용자 데이터를 처리, 저장 또는 전송하지 않습니다. 유일한 데이터 흐름은 다음과 같습니다:
1. 브라우저가 정적 파일 URL을 요청합니다.
2. Cloudflare의 엣지가 캐시된 파일을 제공합니다.
3. 표준 HTTP 로그가 생성됩니다(IP, 타임스탬프, URL, user-agent).

### 데이터는 어디에 저장됩니까?
- **소스 파일:** GitHub 리포지토리(GitHub/Microsoft에 의해 미국에서 호스팅).
- **엣지 캐시:** Cloudflare의 300개 이상 글로벌 PoP. 캐시된 사본은 성능을 위해 전 세계에 분산됩니다.
- **AI 컨시어지 데이터:** 대화는 저장되지 않습니다. 챗 위젯은 메모리 내 세션 상태만 사용하며, 사용자 쿼리에 대한 서버 측 로깅은 없습니다.
- **속도 제한 카운터:** Cloudflare Workers KV에 저장됩니다(집계 카운트만, 개인 데이터 없음).

## GDPR 준수

### 상태
CloudCDN은 GDPR을 준수합니다. 우리는 인프라 제공업체로 Cloudflare를 사용하며, Cloudflare는 다음을 통해 GDPR 준수를 유지합니다:
- EU-US Data Privacy Framework 인증.
- 국제 데이터 전송을 위한 표준 계약 조항(SCC).
- 요청 시 제공되는 데이터 처리 계약.

### 데이터 최소화
- CloudCDN은 쿠키를 설정하지 않습니다.
- 사용자 추적이나 분석 픽셀이 없습니다.
- 개인 데이터는 수집, 저장 또는 처리되지 않습니다.
- HTTP 액세스 로그는 Cloudflare의 개인정보 보호정책에 따라 처리됩니다.

### 정보주체의 권리
CloudCDN은 개인 데이터를 수집하지 않으므로 액세스, 수정 또는 삭제할 개인 데이터가 없습니다. 귀하의 개인 데이터가 실수로 에셋(예: 사진)에 포함되었다고 생각되면, 제거를 위해 support@cloudcdn.pro로 연락하십시오.

### DPA(데이터 처리 계약)
Enterprise 고객은 공식 DPA를 요청할 수 있습니다. sales@cloudcdn.pro로 문의하십시오.

## CCPA / CPRA(캘리포니아)
CloudCDN은 표적 광고를 위해 개인 정보를 판매, 공유 또는 사용하지 않습니다. 개인 데이터를 수집하지 않으므로 옵트아웃 메커니즘이 필요하지 않습니다.

## SOC 2 / ISO 27001
CloudCDN은 다음을 유지하는 Cloudflare의 인프라를 활용합니다:
- SOC 2 Type II 인증.
- ISO 27001 인증.
- PCI DSS Level 1 준수.
이러한 인증은 CloudCDN이 사용하는 엣지 전송 인프라를 다룹니다.

## 보안 조치
- **전송 중 암호화:** 모든 연결에서 TLS 1.3.
- **DDoS 보호:** 모든 플랜에서 Cloudflare의 자동 DDoS 완화.
- **WAF:** Cloudflare 웹 애플리케이션 방화벽이 모든 엔드포인트에서 활성화됨.
- **봇 완화:** Cloudflare Bot Management가 스크래핑 및 남용으로부터 보호합니다.
- **서명된 커밋:** 모든 에셋 변경에는 암호화 검증이 필요합니다.
- **브랜치 보호:** 강제 푸시 및 히스토리 재작성이 차단됩니다.
- **시크릿 관리:** API 토큰은 암호화된 GitHub Secrets로 저장되며, 코드에는 절대 저장되지 않습니다.

## 에셋 무결성
CloudCDN이 제공하는 모든 에셋은 서명된 Git 커밋까지 추적할 수 있습니다. 이는 다음을 제공합니다:
- **출처:** 모든 파일 변경은 검증된 기여자에게 연결됩니다.
- **감사 추적:** 서명된 커밋 검증과 함께 완전한 Git 히스토리.
- **변조 감지:** 무단 수정은 서명 체인을 끊습니다.

## 허용되는 사용
CloudCDN은 정적 에셋 전송 전용입니다. 금지된 사용은 다음과 같습니다:
- 멀웨어 또는 피싱 콘텐츠 호스팅.
- 비디오 스트리밍 또는 대용량 파일 배포(25MB 초과).
- 에셋에 개인 데이터, 자격 증명 또는 민감한 정보 저장.
- 다른 서비스의 약관을 우회하기 위한 서비스 사용.

위반 시 24시간 통지 후 계정이 정지됩니다(불법 콘텐츠는 즉시 제거됨).
