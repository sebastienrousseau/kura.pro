# CloudCDN vs Cloudinary

## 한눈에 보기

| 기능 | CloudCDN | Cloudinary |
|------|----------|------------|
| **시작 가격** | 월 $29 (Pro) | 월 $89 (Plus) |
| **무료 플랜** | 월 10 GB 대역폭 | 월 25 크레딧 (~5K 변환) |
| **요금 모델** | 대역폭 (GB) | 크레딧 기반 (복잡함) |
| **워크플로** | Git 네이티브 (git push) | 대시보드/SDK/API 업로드 |
| **자동 최적화** | 푸시 시 WebP + AVIF + JXL | 즉시 처리만 가능 |
| **AI 기능** | 시맨틱 검색 + RAG 컨시어지 | AI 배경 제거, 생성적 채우기 |
| **서명된 커밋** | 필수 (공급망 무결성) | 해당 없음 |
| **엣지 위치** | 300개 이상 (Cloudflare) | ~60개 CDN PoP |
| **TTFB** | 중앙값 50ms 미만 (NA/EU) | 일반적으로 80-120ms |
| **사용자 정의 도메인** | 5개 (Pro), 무제한 (Enterprise) | 무제한 (유료 플랜) |
| **SLA** | 99.9% (Pro), 99.99% (Enterprise) | 99.9% (Business+) |
| **SSO/SAML** | Enterprise | Enterprise |
| **MCP 서버** | 내장 (AI 에이전트 통합) | 제공되지 않음 |
| **에셋 출처** | 암호화 (서명된 Git 커밋) | 업로드 타임스탬프만 |

## CloudCDN을 선택해야 할 때

- **Git 네이티브 워크플로**를 원할 때 — 대시보드 없음, SDK 없음, 단지 `git push`
- 모든 에셋에 대한 **암호학적 출처**(컴플라이언스, 감사 추적)가 필요할 때
- **단순한 대역폭 청구**(크레딧 계산 없음)를 원할 때
- **정적 에셋**(로고, 아이콘, 배너, 이미지)을 제공하고 50ms 미만의 TTFB를 원할 때
- **AI 기반 에셋 검색**과 에이전트 주도 워크플로를 위한 **MCP 서버**를 원할 때

## Cloudinary를 선택해야 할 때

- **고급 이미지 AI**(배경 제거, 생성적 채우기, 자동 자르기)가 필요할 때
- 적응형 스트리밍을 통한 **비디오 트랜스코딩**이 필요할 때
- 복잡한 이미지 파이프라인을 위한 **400개 이상의 변환 매개변수**가 필요할 때
- SDK가 있는 기존 Cloudinary 통합이 있을 때

## 비용 비교

월 50 GB의 정적 에셋을 제공하는 사이트의 경우:

- **CloudCDN Pro**: 월 $29 정액
- **Cloudinary Plus**: 월 $89 + 크레딧 초과 가능성

월 200 GB의 경우:

- **CloudCDN Pro**: $29 + (100 GB 초과분 × $0.05) = 월 $34
- **Cloudinary Advanced**: 월 $224
